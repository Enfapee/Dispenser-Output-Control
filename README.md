# Dispenser Output Control

A tiny Fabric 1.21.11 client mod prototype that opens a GUI for choosing between two dispenser output items.

The GUI is opened with `O` by default. It lets you type two item IDs, pick slot A or slot B, and save the choice in memory for the current game session.


## Build

```powershell
gradle build
```

The built jar should appear in `build/libs`.

## Requirements

- Minecraft Java Edition 1.21.11
- Fabric Loader 0.19.3 or newer
- Fabric API 0.141.4+1.21.11
- Java 21 recommended
